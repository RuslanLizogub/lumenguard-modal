# Decisions

- 2026-02-10: Обрано мінімалістичну архітектуру з двох модулів: `logic.py` (чиста логіка) + `main.py` (інтеграції/оркестрація).
- 2026-02-10: Поріг циклу валідується на `CHECK_INTERVAL_SECONDS >= 60` для безпечного продакшн-ритму.
- 2026-02-10: Для збереження стану використовується JSON локально та `modal.Dict` у хмарі.
- 2026-02-10: Повідомлення надсилаються тільки при зміні стану, щоб не створювати шум у каналі.
- 2026-02-10: Обрано `parse_mode="HTML"` і `<b>...</b>` для стабільного жирного форматування в Telegram.
- 2026-02-10: Час повідомлення відображається в `TIMEZONE` (default `Europe/Kyiv`).
- 2026-02-10: Тривалість у повідомленні показується лише в годинах і хвилинах (без секунд).
