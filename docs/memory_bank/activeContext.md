# Active Context

## Current State
Базова архітектура LUMENGUARD-MODAL реалізована в мінімальному production-ready варіанті.

## Implemented Flow
1. Таймер (5 хв) запускає цикл.
2. Конфігурація завантажується з `.env`.
3. Для кожної цілі виконується TCP-перевірка доступності.
4. Поточний статус порівнюється з попереднім станом.
5. Якщо статус змінився, рахується тривалість попереднього стану.
6. Генерується українське повідомлення для Telegram.
7. Після успішної відправки стан зберігається.

## Runtime Files
- `logic.py`: чиста бізнес-логіка + JSON persistence helpers.
- `main.py`: оркестрація циклу, `.env`, Telegram, Modal Cron/State bridge.
- `tests/test_logic.py`: базові unit-тести ядра логіки.
